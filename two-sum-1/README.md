# Two Sum（两数之和）

LeetCode 序号: 1 · 题目链接: https://leetcode.cn/problems/two-sum/

问题：给定整数数组 `nums` 和整数 `target`，请在数组中找出两整数，使它们的和为 `target`，并返回它们的索引。每种输入只对应唯一答案，但同一个元素不能被重复使用。

- 输入：`nums = number[]`，`target = number`
- 输出：`[i, j]`（`i <-> j` 无顺序要求）

示例：
- `nums = [2,7,11,15], target = 9` → 输出 `[0,1]`（2+7）
- `nums = [3,2,4], target = 6` → 输出 `[1,2]`（2+4）

边界与约束：
- 至少存在一组可行解（面试常规设定）；元素可能重复，但不能复用同一索引。
- 索引从 0 开始。

核心思路（哈希表）：
- 一次遍历，用哈希 `value -> index`；对当前值 `x=nums[i]`，先查是否存在 `target-x`，若有立即返回两索引；若无再把 `x` 存入哈希。
- “先查后存”避免同一元素被重复使用。

复杂度：
- 时间 O(n)；空间 O(n)

常见陷阱：
- 先存后查会在 `target = 2*x` 时错误复用同一索引。
- 输出顺序不敏感，但测试通常与实现一致即可。
